# System Architecture

## 1. Overview

This document describes the high-level system architecture of a DC Fast Charger (DCFC), covering power conversion, control systems, thermal management, and safety subsystems.

## 5. Thermal Management System

### 5.1 HVAC Clip-On Cooling Unit

The thermal management system uses a modular HVAC unit that clips onto the main charging cabinet, providing forced-air cooling with optional climate control for extreme environments.

```
+------------------------------------------------------------------------------+
|                         THERMAL MANAGEMENT SYSTEM                            |

|  |  +---------------------------+  |    |                                  | |
|  |  |   Compressor/Condenser    |  |    |  +-----------------------------+ | |
|  |  |  (Refrigeration Cycle)    |  |    |  |       Power Modules         | | |
|  |  +-----------+---------------+  |    |  |  +-----+ +-----+ +-----+    | | |
|  |              |                  |    |  |  | M1  | | M2  | | M3  |    | | |
```

### 5.3 HVAC Unit Specifications

| Parameter | Specification | Notes |
|-----------|---------------|-------|
| Cooling Capacity | 15-25 kW thermal | Sized for 350 kW charger |
| Airflow Rate | 2000-4000 CFM | Variable speed for efficiency |

### 5.4 Thermal Control Parameters

```
+-----------------------------------------------------------------------------+
|                       THERMAL CONTROL PARAMETERS                            |
|                                                                             |
|   Component           | Max Temp  | Derating Start | Shutdown Temp          |
|   --------------------+-----------+----------------+-------------------     |
|   SiC MOSFETs         |   175 C   |     150 C      |     165 C              |
|   Inductors           |   130 C   |     110 C      |     125 C              |
|   DC Link Caps        |   105 C   |      85 C      |     100 C              |
|   Transformer         |   140 C   |     120 C      |     135 C              |
|   Cabinet Air (Inlet) |    45 C   |      40 C      |      50 C              |
|                                                                             |
|   HVAC CONTROL MODES                                                        |
|   ---------------------------------------------------------------           |
|   Mode            | Condition              | Action                         |
|   ----------------+------------------------+--------------------------      |
|   Idle            | No charging session    | Fans off, compressor standby   |
|   Pre-Cool        | Session starting       | Ramp up cooling before load    |
|   Active Cooling  | Charging in progress   | Modulate fan/compressor        |
|   Max Cooling     | Temp approaching limit | Full compressor + max fan      |
|   Derating        | Cooling insufficient   | Reduce power to protect        |
|                                                                             |
+-----------------------------------------------------------------------------+
```

### 5.5 Benefits of Clip-On HVAC Design

| Benefit | Description |
|---------|-------------|
| **Modularity** | HVAC unit can be serviced or replaced without opening main cabinet |
| **Scalability** | Different HVAC sizes for different power levels (50kW, 150kW, 350kW) |
| **Climate Flexibility** | Optional heater integration for cold climate operation |
| **Simplified Cabinet** | Main cabinet design simplified without integrated cooling |
| **Field Upgradability** | HVAC can be upgraded independently of power electronics |
| **Reduced Downtime** | Quick-swap capability for maintenance |
