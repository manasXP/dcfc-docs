# Cabinet Layout - DC Fast Charger

Tags: #dcfc #cabinet #layout #mechanical #enclosure

Related: [[__Workspaces/ECG/__init]] | [[01 - Hardware Components]] | [[02 - Electric Wiring Diagram]]

## 1. Overview

This document provides the physical cabinet layout for a 150-350 kW DC Fast Charger terminal, including component placement, dimensions, thermal zones, and cable routing specifications.

## 2. Cabinet Specifications

### 2.1 Dimensions

| Parameter | Value | Notes |
|-----------|-------|-------|
| Overall Height | 1800 mm | Including pedestal |
| Overall Width | 700 mm | |
| Overall Depth | 550 mm | |
| Pedestal Height | 200 mm | Concrete/steel base |
| Main Cabinet Height | 1600 mm | |
| Weight (empty) | 150 kg | Cabinet only |
| Weight (loaded) | 450-600 kg | Full system |

### 2.2 Environmental Ratings

| Parameter | Rating |
|-----------|--------|
| IP Rating | IP54 (outdoor) |
| IK Rating | IK10 (front panel) |
| NEMA Rating | 3R / 4X |
| Operating Temperature | -30°C to +50°C |
| Storage Temperature | -40°C to +70°C |
| Humidity | 5-95% non-condensing |
| Altitude | Up to 2000m |

### 8.1 Thermal Management Strategy

| Zone | Temperature Range | Cooling Method | Components |
|------|------------------|----------------|------------|
| Zone 1 | 35-45°C | Forced air | Controller, displays, communication |
| Zone 2 | 55-70°C | Liquid cooling | Power modules, SiC MOSFETs |
| Zone 3 | 40-50°C | Liquid + air | Pump, reservoir, heat exchanger |
| Zone 4 | 35-50°C | Natural convection | Circuit breakers, contactors, meter |

### 9.1 Cable Routing Guidelines

| Cable Type | Route | Separation | Notes |
|------------|-------|------------|-------|
| AC Power (L1, L2, L3, N) | Right cable tray | 200mm from control cables | Use cable ties every 300mm |
| DC Output | Center/front | Shielded, separate tray | Liquid-cooled above 150kW |
| Control (24V DC) | Left cable tray | Away from power | Screened cables |
| Communication (CAN, ETH) | Left cable tray | 100mm from power | Shielded twisted pair |
| Coolant Lines | Rear/center | Flexible routing | Insulated tubing |
